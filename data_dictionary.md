# Data Dictionary

This document describes the columns used in the sales dataset.

| Column Name | Data Type | Description |
|-----------|----------|-------------|
| date      | DateTime | Date of transaction |
| product   | String | Name of the product |
| category  | String | Product category |
|price_per_kg | Float | Price per kilogram |
| weight_kg | Float | Quantity sold in kilograms |
| revenue   | Float | Total revenue from the sale |
